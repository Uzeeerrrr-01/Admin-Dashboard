## 📌 Project Title

**CourseHub – Course Management System**

---

## 📖 Description

CourseHub is a full-stack web application designed to simplify the management of courses and instructors. It allows users to create, view, update, and delete course data while maintaining a structured and user-friendly interface. The platform is built to help educational institutions, instructors, or learners efficiently organize and access course-related information.

---

## 🚀 Project Overview

CourseHub provides a centralized system where:

* 📚 Courses can be managed easily
* 👨‍🏫 Instructor information is maintained
* 🔄 Real-time updates are reflected in the UI
* 🌐 Frontend and backend communicate seamlessly

The application follows a **MERN stack architecture** ensuring scalability and performance.

---

## 🛠️ Tech Stack

**Frontend:**

* React (Vite)
* Tailwind CSS

**Backend:**

* Node.js
* Express.js

**Database:**

* MongoDB (MongoDB Atlas)

---

## ✨ Features!

* ✅ Add new courses
* ✅ View all courses
* ✅ Update course details
* ✅ Delete courses
* ✅ Manage instructors
* ✅ RESTful API integration
* ✅ Responsive UI

---

## 🔗 API Endpoints

### Courses

* `GET /courses` → Get all courses
* `POST /courses` → Create a course
* `PUT /courses/:id` → Update course
* `DELETE /courses/:id` → Delete course

### Instructors

* `GET /instructors` → Get all instructors
* `POST /instructors` → Add instructor

---

## ⚙️ How It Works

1. User interacts with the frontend UI
2. React sends requests to the backend API
3. Express server processes the request
4. Data is stored/retrieved from MongoDB
5. Response is sent back and displayed in UI

## 📦 Installation & Setup

```bash

# Install frontend dependencies
cd frontend

# Install backend dependencies
cd backend
npm install
npm run server
```
